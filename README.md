Firmware code and Hardware schematics

## Quick Start

Install PlatformIO

```shell
$ make build
```

## Upload Firmware

Compatible hardware is required. 
PlatformIO will attempt to auto-discover the correct com port.

```shell
$ make upload
```

## Tests

```shell
$ make test
```
